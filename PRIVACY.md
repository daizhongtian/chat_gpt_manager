# Privacy Policy for ChatGPT Cleaner & Context Viewer

Last updated: June 2026

ChatGPT Cleaner & Context Viewer is a Chrome extension designed to help users manage visible ChatGPT conversations and estimate visible page context usage locally on chatgpt.com.

## Single Purpose

The single purpose of this extension is to help users manage visible ChatGPT conversations and estimate visible page context usage locally in their browser.

## Data Handled by the Extension

The extension may process the following data locally in the user's browser:

- Visible ChatGPT conversation content on the current page
- Visible ChatGPT sidebar conversation information
- User actions inside the extension, such as selecting conversations, deleting selected conversations, refreshing the visible conversation list, estimating context usage, and using the local usage counter
- Local extension settings, such as whether the extension is enabled, whether context estimation is enabled, the selected context window size, and local usage count values

## How the Data Is Used

The extension uses this data only to provide its user-facing features:

- Adding checkboxes to visible ChatGPT sidebar conversations
- Letting users select visible conversations
- Deleting selected conversations through ChatGPT's normal delete confirmation flow
- Refreshing the visible conversation list
- Estimating visible context usage on the current ChatGPT page
- Counting local model usage in the browser

## Data Storage

The extension stores settings and usage counters locally in the user's browser using Chrome extension storage.

The extension does not store ChatGPT conversation content on an external server.

## Data Sharing and Transfer

The extension does not sell, transmit, or share user data with third parties.

The extension does not send ChatGPT conversation content, account information, personal data, browsing activity, or usage counters to any external server.

## Remote Code

The extension does not use remote code. All JavaScript, CSS, and executable resources are packaged inside the extension.

## Limited Use

The extension uses data only to provide or improve its single purpose and user-facing features. The extension does not use user data for advertising, analytics, profiling, resale, creditworthiness, lending, or unrelated purposes.

## Third-Party Services

The extension is not affiliated with, endorsed by, or officially connected to OpenAI or ChatGPT.

## Contact

For questions about this privacy policy, please contact the developer through the Chrome Web Store listing or the project repository.
